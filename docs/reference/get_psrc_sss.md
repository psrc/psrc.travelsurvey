# Retrieve PSRC transportation safety & security survey data

Retrieve PSRC transportation safety & security survey data

## Usage

``` r
get_psrc_sss(survey_vars = NULL)
```

## Arguments

- survey_vars:

  desired variables (in addition to person and household ids plus
  \`person_weight\`, which are returned by default)

## Value

data.table of queried safety survey responses
