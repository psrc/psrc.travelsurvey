# Retrieve PSRC travel survey data

Retrieve PSRC travel survey data

## Usage

``` r
get_psrc_hts(survey_years = c(1719, 2017, 2019, 2021, 2023, 2025), survey_vars)
```

## Arguments

- survey_years:

  vector with values that may include 1719, 2017, 2019, 2021, 2023, 2025

- survey_vars:

  desired variables (in addition to id fields, which are returned by
  default)

## Value

surveyDataTools data object, i.e. named list
